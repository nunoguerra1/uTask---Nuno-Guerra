## Descrição do Projeto

O projeto **uTask 3.0** consiste em uma aplicação que simula um quadro Kanban, permitindo ao usuário adicionar tarefas e movimentá-las entre as colunas **“A Fazer”**, **“Em Andamento”** e **“Concluído”**. Além disso, a interface de dashboard exibe uma frase motivacional, e o sistema oferece funcionalidades de cadastro e autenticação de usuários.

### Resumo das Principais Tecnologias

* **Interface Moderna com React:** A aplicação é desenvolvida utilizando React, garantindo uma experiência rápida, responsiva e dinâmica para o usuário.
* **Tipagem Segura com TypeScript:** Todo o código é estruturado em TypeScript, proporcionando maior confiabilidade, produtividade e redução de erros durante o desenvolvimento.
* **Back-end Simulado com JSON Server:** O sistema utiliza JSON Server para simular uma API completa, permitindo operações de cadastro, login e manipulação de tarefas sem a necessidade de um servidor real.
* **Quadro Kanban Dinâmico:** A aplicação conta com um quadro Kanban totalmente interativo, onde as tarefas podem ser organizadas entre as colunas por meio da funcionalidade de drag and drop, facilitando a visualização e o gerenciamento do fluxo de trabalho.
